Starting

Oct 21st, 2012

I was very in doubt which tool to use for blogging tech things, but finally I’m using [Octopress|http://octopress.org/]. I like Markdown, Vim, Git too much to use web based blogging service. Octopress is best choise I had ever seen for such user.

  • Fully customized if you want with writing ruby code or adding plugins.
  • “import” or “export” is NOT necessary: all of articles built from Markdown files in your local filesystem.

To setup:

git clone https://github.com/imathis/octopress.git
cd octopress
bundle install          # you should `gem install bundler` before!
rake setup_github_pages # need your repository url
# edit your _config.xml file to set url, title, author, etc.
rake gen_deploy
# you can see your blog on http://USERNAME.github.com/
rake new_post[title]    # \[title\] for zsh
# edit your new post on sources/_posts/*.markdown
rake gen_deploy
# YOU MADE IT!!!

Enjoy happy tech bloggin’ ;)
